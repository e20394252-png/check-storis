The Challenge
Как проверить репост
без API Instagram?
Нужна была система регистрации на мероприятие, где условием участия служит репост сторис в Telegram. Без публичного API — только скриншот как доказательство, ручная проверка администратором, и мгновенный фидбек пользователю.
The Solution
  • Telegram Mini App — нативный интерфейс прямо в мессенджере, авторизация через initData без регистрации
  • Foto-first модерация — скриншот приходит в личку бота с кнопками, не нужно открывать веб-панель
  • Мгновенный фидбек — пользователь сразу видит статус: на проверке → одобрено → бейдж участника
  • Railway CI/CD — push в GitHub → автодеплой за 3 мин, PostgreSQL managed
